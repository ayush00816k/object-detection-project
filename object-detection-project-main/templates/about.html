    • About The Project

    🎯

    Research Objective

    Conduct a comprehensive comparative analysis of ML & DL models for object detection, evaluating their performance across multiple metrics.

    📊

    Performance Evaluation

    Assess accuracy, precision, and computational efficiency of YOLO, CNN, ResNet-18, MobileNet, KNN, SVM, and Ensemble methods.

    🧠

    Methodology

    Utilize standardized datasets and high-performance testing environments to ensure fair comparison between statistical and neural approaches.

    Deep Learning Models

    YOLO (You Only Look Once)

    A state-of-the-art, real-time object detection system that frames detection as a regression problem, achieving remarkable speed in a single pass.

    CNN (Convolutional Neural Network)

    Custom-built 4-block architecture designed for hierarchical feature extraction from grid-like image data topology.

    ResNet-18

    Residual Neural Network architecture utilizing skip connections to solve vanishing gradients in deep training cycles.

    MobileNet

    Lightweight architecture optimized for mobile and embedded vision using depthwise separable convolutions.

    Machine Learning Models

    KNN (K-Nearest Neighbors)

    Instance-based learning algorithm that classifies objects based on the proximity in feature space.

    SVM (Support Vector Machine)

    High-dimensional hyperplane optimization used for robust class separation and linear/non-linear classification.

    Decision Tree

    Tree-structured model utilizing hierarchical entropy reduction for interpretable rule-based detection.

    Random Forest

    Ensemble approach constructing multiple decision trees to reduce overfitting and improve collective accuracy.
